## Checkout Order

Reichere Order um Checkoutdaten an

### URL
`/v1/orders/:orderid/checkout`

### Method
`POST`

### Accept
`application/json`

### Content-Type
`application/json`

### URL Params
`orderId=[long]`

### Data Params
Order als JSON
